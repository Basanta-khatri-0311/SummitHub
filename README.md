# 🏔️ SummitHub

A community-driven platform for trekkers to share routes, experiences, and connect with fellow adventurers.

## 🚀 Overview
SummitHub is designed to be the go-to platform for trekking enthusiasts. Users can explore real trekking experiences, discover routes, and connect with others planning similar adventures.

---

## ✨ Features (MVP)
* **🧑‍🤝‍🧑 User Authentication:** Secure signup and login functionality.
* **📝 Share Trek Experiences:** Document and post your journey stories.
* **📸 Upload Trek Images:** Share high-quality visuals from your trips.
* **❤️ Like & Comment on Posts:** Social engagement within the community.
* **👤 User Profiles:** Manage your personal trekking portfolio.

## 🔮 Upcoming Features
* **🗺️ Trek Route Mapping:** Full GPX support and interactive maps.
* **🤝 Trek Partner Finder:** Find companions for upcoming treks.
* **🏆 Leaderboards & Badges:** Gamified rewards for active contributors.
* **📍 Smart Trek Discovery:** Recommendations based on difficulty and location.
* **📥 Offline Trek Data:** Access routes without an internet connection.

---

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

## 📂 Project Structure
* `client/` → Frontend (React)
* `server/` → Backend (Node/Express)

---

## ⚙️ Setup Instructions

### 1. Clone the repo
```bash
git clone [https://github.com/your-username/summithub.git](https://github.com/your-username/summithub.git)
cd summithub
```

### 2. Setup Backend
```bash
Bash
cd server
npm install
npm run dev
```


### 3. Setup Frontend
```bash
Bash
cd client
npm install
npm start
```
